## Разработка

После клонирования проекта и установки зависимостей с помощью `bun i`, запустите сервер разработки:

> Мы используем _[bun](https://bun.com)_ для разработки

```bash
bun dev

# или можно одновременно еще открыть вкладку в браузере
bun dev -- --open
```

## Сборка прода

```bash
bun run build
```

> Вы можете предварительно просмотреть собранное приложение с помощью `bun preview`.
